How to use this tool

  • Begin by clicking the 'BEGIN THE ASSESSMENT' button below.
  • Go through each section using the 'NEXT' button on each page.
  • Each section contains 2 to 5 questions. Use the dropdown menus to select the option that best fits your knowledge.
  • Select answers based on where your organisation is now and where you want to be in the future (3-5 years).
  • Your final score is based on weighted performance across all sections, resulting in a level of maturity and a tailored action plan.
Level 1: Observer Level 2: Early Adopter Level 3: Effective Deliverer Level 4: Leader Level 5: Innovator

You can also view the PHOCUS Maturity Reference Model, which this tool is based upon, or access your results and action plan once the assessment is complete.

Note: This is a prototype for testing purposes. Your data will not be recorded.
